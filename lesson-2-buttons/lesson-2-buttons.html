Anchors (links)

These are links to other pages.

We need to create an "a" tag.

Ez egy speciális tag az eddigiekhez képest. Meg kell adnunk neki egy attribute-t. Magyarul leggyakrabban attribútumnak hívjuk. This is a special tag. We need to add a special attribute: href (hypertext reference).

The attribute is in the opening tag. It is a reference to the target page where we want to navigate.
